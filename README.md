# 陳孟潔編劇作品集網站

Static portfolio website draft with a cinematic A24 + Netflix-inspired direction.

## Structure

- `index.html` - page shell and semantic sections
- `assets/css/styles.css` - visual system and responsive layout
- `assets/js/main.js` - carousel, language toggle, dynamic rendering
- `data/site-content.js` - bilingual public content and asset references
- `data/source-*.html` - local reference snapshots from public source pages
- `docs/private-brief.md` - private user-supplied profile notes not shown publicly
- `docs/sources.md` - source list for verification

## Run Locally

Open `index.html` directly, or run a local server:

```sh
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.
